    **Step 3: Create workout routes**

    Create ../backend/app/workouts/routes.py with router prefix "/workouts", tags ["workouts"]. Endpoints:

    1. `POST /workouts` (WorkoutCreate -> WorkoutStartResponse, 201): Verify program exists (404 if not). Create Workout row with program_id. Compute pre_fill: for each exercise in the program, find the most recent completed workout that has sets for that exercise_id, return those sets' actuals. If no prior session, use program template targets (target_reps -> reps, target_weight_kg -> weight_kg). Return the workout + pre_fill map.

    2. `GET /workouts/active` (-> WorkoutRead, 200 or 404): Query Workout where completed_at IS NULL, ordered by started_at desc, first(). Use selectinload for sets -> exercise. Return 404 if none.

    3. `GET /workouts/{workout_id}` (-> WorkoutRead, 200): Eager load with selectinload. 404 if not found.

    4. `PATCH /workouts/{workout_id}/complete` (-> WorkoutRead, 200): Set completed_at = datetime.now(UTC). Return updated workout.

    5. `POST /workouts/{workout_id}/sets` (WorkoutSetCreate -> WorkoutSetRead, 201): Create WorkoutSet row. Eager-load exercise on the created set for the response (query it back with selectinload or just join).

    6. `PUT /workouts/{workout_id}/sets/{set_id}` (WorkoutSetUpdate -> WorkoutSetRead, 200): Update weight_kg and/or reps. Return updated set with exercise.

    7. `DELETE /workouts/{workout_id}/sets/{set_id}` (204): Delete the set. 404 if not found.

    8. `DELETE /workouts/{workout_id}/exercises/{exercise_id}` (204): Delete all WorkoutSet rows matching workout_id + exercise_id. Return 204 (even if zero sets matched -- idempotent).

    9. `DELETE /workouts/{workout_id}` (204): Delete the workout (cascade deletes sets). 404 if not found.

    Also add settings endpoints (can be on same router or a separate one registered in main.py):

    10. `GET /settings/{key}` (-> SettingRead, 200): Return setting or 404.

    11. `PUT /settings/{key}` (SettingUpdate -> SettingRead, 200): Upsert (create or update) the setting. Return it.
