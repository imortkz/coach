    - get_weight_unit returns "kg" when no setting exists, returns "lb" when setting set
    - convert_weight(100, "lb") returns 220.5, convert_weight(100, "kg") returns 100
    - get_1rm_data with a session containing workout sets returns list of dicts with exercise, current_1rm, best_1rm, trend keys
    - 1RM uses Epley formula: weight * (1 + reps/30). For weight=100, reps=5: 1RM = 100 * (1 + 5/30) = 116.67
    - Warmup sets (is_warmup=True) are excluded from 1RM calculation
    - Sets with None weight_kg or None reps are excluded
    - Exercises with fewer than 2 sessions show trend as None/"-"
    - Trend: compare average 1RM of last 3 sessions vs previous 3 sessions. Less than 2% change = "flat", higher = "up", lower = "down"
    - get_volume_data returns list of dicts with muscle_group, total_sets, total_volume_kg, percentage keys
    - Volume = sum of (weight_kg * reps) per set, grouped by muscle_group. Warmup excluded.
    - Percentages sum to 100 (or close with rounding)
    - get_frequency_data returns weekly breakdown of workouts. Only completed workouts (completed_at not None) counted.
    - All query functions respect period_days parameter for date filtering
    - Empty data returns empty lists (no crashes)
  </behavior>
